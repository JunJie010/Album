## 项目职责
* 通过Linux Framebuffer机制，mmap内存映射，调用文件IO函数**在开发板上显示图片**
* 利用Linux输入子系统，捕获解析struct input_event，**实现基于坐标判别的触摸交互**
***
**本项目是一个典型的三层嵌入式GUI应用**
* 硬件层：LCD显示设备，触摸屏
* Linux内核层：通过设备文件(/dev/fb0, /dev/input/event0)暴露硬件接口
* 应用层：直接操作底层设备的C语言程序
